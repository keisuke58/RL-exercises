Level 3 Observations: Contextual MarsRover
==========================================

Setup
-----
Context features (2 features):
  - goal_reward : reward at rightmost state (position 4)
  - slip_prob   : probability that chosen action is flipped (1 - P[s,a])

Training contexts : [(5.0, 0.0), (10.0, 0.2), (15.0, 0.4)]
Validation contexts : same as training (Section 6.4 recommendation)
Test contexts (interpolation) : [(7.5, 0.1), (12.5, 0.3)]
Test contexts (extrapolation) : [(20.0, 0.6), (2.0, 0.5)]

Context rotation during training: round-robin over training contexts.

"With context"    : VI computes optimal policy per context using that context's MDP.
"Without context" : VI runs on averaged training MDP; single blind policy for all contexts.

Results
-------
Validation Set:
  Context               With context   Without context
  goal=5.0  slip=0.0        45.00           45.00
  goal=10.0 slip=0.2        61.51           61.51
  goal=15.0 slip=0.4        45.35           45.35

Test (Interpolation):
  goal=7.5  slip=0.1        57.16           57.16
  goal=12.5 slip=0.3        56.05           56.05

Test (Extrapolation):
  goal=20.0 slip=0.6        32.23           21.23  ← clear gap
  goal=2.0  slip=0.5         5.42            5.56

Observations
------------
1. Validation and interpolation: both methods perform identically.
   The averaged training MDP yields policy [1,1,1,1,1] (always right),
   which is also optimal for all in-distribution and interpolation contexts.
   The context-aware agent agrees on this policy, so no gap appears.

2. Extrapolation (goal=20.0, slip=0.6): context-aware wins (+11 reward).
   With slip_prob=0.6, actions are unreliable. The context-aware VI adapts
   its policy to this high-noise MDP and finds a better strategy.
   The context-blind agent applies the policy from the low-noise training
   distribution, which is suboptimal here.
   This matches the paper's finding that context helps most at extrapolation.

3. Extrapolation (goal=2.0, slip=0.5): both methods nearly tied.
   When slip_prob=0.5, all actions are equivalent (T[s,a,:] is identical
   for both actions at all interior states), making the environment a
   random walk. No policy can consistently reach the right end, so both
   methods perform similarly regardless of context.

4. Where results align with the paper (Section 6.4):
   - Providing context helps on out-of-distribution (extrapolation) contexts.
   - Interpolation generalizes well even without context.
   - The gap between context-aware and blind is largest when the new context
     changes the optimal policy structure (high slip → different strategy).

5. Where results differ from the paper:
   - The paper shows larger and more consistent gaps due to complex environments
     where context changes which actions are optimal across many states.
   - In MarsRover, the effect is visible only at extreme extrapolation because
     the 1D structure limits how much context can alter optimal behavior.
   - Deep RL methods in the paper suffer from sample inefficiency on top;
     our model-based VI solves each context exactly given T and R.
